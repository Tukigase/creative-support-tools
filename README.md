# 🛠️ Creative Support Tools

クリエイター、デザイナー、エンジニアの「作業中のちょっとした手間」をブラウザ上で一瞬で解決する、ニッチで実用的なWebツール群です。
すべてサーバー通信なし（完全クライアントサイド動作）でサクサク軽量に動きます。

🌍 **[サイトを見る (GitHub Pages)](https://tukigase.github.io/creative-support-tools/)**

## 📦 収録ツール
現在公開されているツールの一覧です。（今後も機能・ツール追加を予定しています）

* **🎨 colorPicker（配色テスター）**
  * 背景色と文字色の組み合わせを直感的に確認できます。
  * HEX / RGB 値のワンクリックコピー。
  * WCAG準拠のコントラスト比判定と色覚シミュレーション付き。

* **📐 aspectRatio（アスペクト比計算機）**
  * 画像や動画のサイズを指定した縦横比に自動計算。
  * CSSで使えるボックスサイズやハックパターンを出力。

* **🧩 colorPalette（カラーパレット生成）**
  * 7つのカラースロットをリアルタイムで編集・プレビュー。
  * CSS変数とMarkdown用カラー表を自動生成。
  * `:root`形式のCSSインポートもサポート。

* **🔤 fontTester（フォント比較テスター）**
  * 複数Webフォントを同時に比較表示。
  * 文字組み・字間・行間を確認しながら最適な候補を検討できます。

* **📏 modularScale（モジュラースケール計算機）**
  * ベースフォントサイズと比率を選ぶだけで、見出しから本文までの数値を自動算出。
  * CSS用のサイズ一覧をそのままコピー可能。

* **📝 markdownEditor（Markdownエディター）**
  * Markdownをリアルタイムプレビュー。
  * Mermaid記法対応で図表をブラウザ内レンダリング。
  * Markdown / PNG / PDF のエクスポート機能付き。

## 📐 UI / レイアウト仕様
* **ツールカード画像 (サムネイル):** アスペクト比 `16:9` で統一
* **ツールカード本体:** アスペクト比 `1 : 1.4141`（白銀比）、幅 `300px` 基準のレスポンシブ配置

## 💻 技術スタック
* HTML5 / CSS3 / Vanilla JavaScript
* **Jekyll** (GitHub Pagesを利用したレイアウト・コンポーネント管理)
* 外部フレームワーク不使用（軽量・高速）
